## README

This url shortener let's you shorten any urls and track them. Feel free to fork and cusomize it for yourself.

Things to keep in mind: 

* Ruby on Rails app
* I used [Devise](https://github.com/plataformatec/devise) for user registration
* I used [Simple Form](https://github.com/plataformatec/simple_form) for my forms

That's all!
